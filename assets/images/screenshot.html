HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Only use h1 element once per webpage, and use the rest of the h(#) elements in order
  • The ul element creates an unordered list
  • Wrap each list item in an li element
  • HTML creates the structure of the website
  • The head element contains information, or metadata, that helps the browser to render the page correctly
  • Elements included within the head are not visible when a user looks at a webpage

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The external CSS stylesheet is most commonly used by developers because it allows CSS rules to be kept in seperate files
  • When using the external stylesheet we must link the files with the link element to create a connection to the external file, in the head element, under metadata
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations have two components, the CSS property we want to apply and the value for that property
  • We can use a "*" to declare a rule is applied to all elements on the page
  • Two or more elements that will share the same styling can be grouped together
  • text-align is a positioning property
  • img selector assigns property values to any image element in the markup
  • In linked files, the rel attripute specifies the relationship between the current document and the linked document source
  • The href attribute specifies the location of the external recource
  • class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • The "." in the ".card" command designites this as a class selector, so any HTML element assigned to the class card will gain these property assignments from CSS
  • Hex #'s or semantic refferences can be used to aaign colors

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and swithces to it
  • git checkout main swithces to the main branch
  • feature brannches make it so multiple developers can work on one branch at the same time without problems
  • It is important to frequently synch an updated version of your code to github
  • git add -A command is used after you save a file on VS code
  • git commit -m "description" commits your file to github.com
  • git pull origin main pulls in the base branch
  • git push origin (feature/name of branch) pushes up all of the changes
  • its a good habit to always pull the base branch in your feature before pushing any code

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes script
  • After assigning a value to a variable you can change it later in the code
  • A string is a sequence of text, and we use "" to enclose strings
  • A number is a number
  • A Boolean is a true or false statement
  • An array is a structure which allows you to store multiple values in a single reference
  • An object can be anything, everything in JavaScript is an object and can be stored in a variable
  • The '=' assigns a value to a variable
  • Strict equality or '===' performs a test to see if two values are equal& returns a Boolean result
  • Not/ Does-not Equal: '!' and '!==' return the logically opposite value of what it percedes
  • Conditionals are code structures used to test if an expression returns true or not
  • Creating a function defines a block of code that performs a specific task given a unique name
  • You must call a function in your code to execute it
  • The script.js file was linked with the body element on the index.html file
  • Arrays typically hold data that is related in some way
  • Arrays are useful when we want to perform a task on multiple pieces of data at once
  • To locate a specific data item in an Array you need the index #
  • Index numbers start at 0
  • The steps for creating a loop include: (1) Determine the starting point of the loop (2) Determine the condition for the loop (3) Determine what allows the array to iterate over each item (4) include the conditional statement in the code block {}
  • The command: var randomTopic = topics[Math.floor(Math.random() * topics.length)]; enables the computer to generate a randomized result in the code
