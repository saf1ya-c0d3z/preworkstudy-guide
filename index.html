>HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Only use h1 > element once per webpage, and use the rest of the h(#) elements in order
  • The ul element creates an unordered list
  • Wrap each list item in an li element
  • HTML creates the structure of the website
  • The head element contains information, or metadata, that helps the browser to render the page correctly
  • Elements included within the head are not visible when a user looks at a webpage

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and swithces to it
  • git add -A command is used after you save a file on VS code
  • git commit -m "description" commits your file to github.com
  • git checkout main swithces to the main branch
  • feature branches make it so multiple developers can work on one branch at the same time without problems
  • It is important to frequently synch an updated version of your code to github
  • git pull origin main pulls in the base branch
  • git push origin (feature/name of branch) pushes up all of the changes
